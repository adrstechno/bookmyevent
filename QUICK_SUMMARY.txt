FEATURE BREAKDOWN
================================================================================

FREE TRIAL:
  - Duration: 90 days
  - Auto-created on vendor registration
  - Shows countdown in dashboard
  - Warns at 7, 1 days before expiry
  - Blocks full feature access after expiry

PREMIUM PLAN:
  - Price: ₹499/year
  - Payment: Razorpay integration
  - Duration: 365 days
  - Unlocks all features
  - Overrides trial restrictions

FEATURE RESTRICTIONS:
  Free User CANNOT see:
    - Customer name, email, phone
    - Full booking address
    - Special requirements
    - Payment amount
    
  Free User CAN see:
    - Booking date/time
    - Service type
    - Location (general)
    - Status

  Premium User:
    - Sees everything
    - No restrictions
    - Full analytics
    - Multiple locations
